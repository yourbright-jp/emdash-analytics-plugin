# EmDash Analytics Plugin

Google Search Console and GA4 analytics plugin for EmDash.

This package provides:

- site-wide analytics sync for public pages
- opportunity scoring for managed content
- dashboard widgets and admin pages
- read-only agent API with plugin-scoped API keys

## Status

Initial implementation for YourBright. Publish automation is scaffolded, but npm credentials still need to be configured in GitHub Actions before public release.

## Temporary install

Until npm publishing is enabled, install from GitHub:

```json
{
  "dependencies": {
    "@yourbright-jp/emdash-analytics-plugin": "github:yourbright-jp/emdash-analytics-plugin"
  }
}
```
